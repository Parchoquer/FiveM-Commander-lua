# FiveM Commander lua
# Currently in development
Hi, Fivem Commander is a small bot that allows you to control your server **via Discord**.

## What is he doing
- Great commands to manage your server

## Install
- Downlaod and install FiveM-Commander-lua into your resources added `ensure FiveM-Commander-lua` in server.cfg
- Complete the configuration file
- Restart server and test command `ping`

## For developers
